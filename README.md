# Nova AI Assistant 
Nova AI is a Python-based voice-controlled AI assistant that can listen to user commands and perform various tasks such as answering questions, searching the internet, opening applications, and interacting with the system through voice.

## Features
- Voice recognition using speech input
- AI chatbot responses
- Real-time web search
- Open applications automatically
- GUI interface built with PyQt5
- Text-to-speech responses
- Automation of common tasks

## Technologies Used 
- Python
- PyQt5
- Selenium
- Speech Recognition
- Text to Speech (pyttsx3)
- Webbrowser
- AppOpener
